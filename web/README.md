# NLW #3 // Happy - Web

## Init

Instale as dependências com `yarn install` e rode o projeto usando `yarn start`.

Se tudo estiver ok com a instalação, a aplicação poderá ser acessada em `localhost:3000`.

## Tools

- React
- Leaflet
